System Architecture

The system follows a cloud-native architecture pattern, designed for scalability and high availability.

High-Level Enterprise DevOps Architecture

Key Components

  • Infrastructure: Provisioned using Terraform (VPC, Subnets, EC2).
  • Orchestration: Docker Compose for multi-container web and monitor services.
  • Monitoring: Real-time health checks and alerting via Bash automation.
